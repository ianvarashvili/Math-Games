
This is the backend server for a Georgian-language math games website.

### Technologies Used

- Python + Flask
- Firebase Firestore
- Firebase Authentication

### Setup

1. Install dependencies: `pip install -r requirements.txt`
2. Set up Firebase credentials in `serviceAccountKey.json`
3. Configure environment variables in `.env` (e.g., `FIREBASE_API_KEY`)
4. Run the server: `python app.py`
